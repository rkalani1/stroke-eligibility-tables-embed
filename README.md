# Stroke Eligibility Tables Public Reference Demo

This repository contains static HTML reference tables for stroke study eligibility. The public GitHub Pages build is a demo and code preview only.

Live preview: https://rkalani1.github.io/stroke-eligibility-tables-embed/

## Compliance Guardrails

- Do not iframe the personal GitHub Pages URL into any organization, EHR, intranet, SharePoint, patient-facing, or recruitment pages without approval.
- Do not treat this repository as the source of truth for active trial eligibility, contact workflows, or recruitment instructions.
- Before operational use, verify the current table content against the study-owner-approved source, IRB/HSD-approved materials where applicable, and the approved institutional publishing location.
- This repository must not contain patient identifiers, PHI, research participant information, or confidential operational data.

## Approved-Use Path

If these tables are useful for a real clinical, recruitment, or department workflow, move the approved content to a approved system with named ownership, last-reviewed date, update cadence, and security/governance review.
